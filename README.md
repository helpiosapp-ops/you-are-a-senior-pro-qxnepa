
# ExitPlanner

A calm, offline, privacy-first app that helps professionals prepare thoughtful exits from their jobs.

## Overview

ExitPlanner is designed for professionals who need to prepare for career transitions—whether planning a resignation, preparing for potential layoffs, negotiating an exit, or taking a career break.

## Key Features

- **Four Exit Paths**: Resignation, Layoff Risk, Negotiated Exit, Career Break
- **Guided Checklists**: Step-by-step preparation items for each scenario
- **Mistake Avoidance**: Learn common pitfalls to avoid
- **Key Considerations**: Important factors to think about
- **Personal Notes**: Private space for your thoughts and planning
- **PDF Export**: Generate professional exit plan documents
- **Complete Privacy**: All data stored locally, no cloud sync

## Privacy & Security

ExitPlanner is built with privacy as the foundation:

- ✅ No user accounts or registration
- ✅ No cloud synchronization
- ✅ No analytics or tracking
- ✅ No ads or in-app purchases
- ✅ All data stored locally on device
- ✅ No external API calls
- ✅ No third-party integrations

Your data never leaves your device.

## Technical Stack

- **Framework**: React Native + Expo 54
- **Language**: TypeScript
- **Storage**: AsyncStorage (local only)
- **PDF Generation**: expo-print
- **Sharing**: expo-sharing
- **Navigation**: expo-router
